BioGraph es una herramienta que usa grafos para modelar como interactuan las proteinas de bacterias patogenas y asi ayudar a descubrir nuevos farmacos.
El sistema permite cargar datos desde archivos CSV, modificar la red quitando protenas  y visualizar los complejos usando algoritmos como BSF, DFS y Dijkstra.
Es un proyecto interactivo desarrollado en Java con una interfaz grafica que ayuda a identificar cuales son los puntos debiles de un patogeno para atacarlos eficazmente
